Проект: Сервис обмена сообщениями
Технологии:
- FastAPI
- PostgreSQL
- Redis
- SQLAlchemy
- Celery
- Docker
- Aiogram
Установка и запуск:
Требования:
- Docker
- Docker Compose
1. Клонируйте репозиторий:
   ```bash
   git clone <ссылка на репозиторий>
   cd <название проекта>
2.Запуск контейнеров
   docker-compose up -
3.Выполните миграции
  docker-compose exec web alembic upgrade head
Проект готов к работе!
