Установка и запуск:
Требования:
Docker
Docker Compose
Шаги для запуска:
Клонируйте репозиторий:
git clone https://github.com/spiculor/MindFusion
cd MindFusion

Запустите контейнеры:
docker-compose up

Выполните миграции базы данных:
docker-compose exec web alembic upgrade head

Теперь проект готов к работе!
